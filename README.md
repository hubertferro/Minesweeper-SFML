# Minesweeper-SFML

A C++ Minesweeper game built with SFML. This project focuses on game logic, tile rendering, event handling, and object-oriented programming.

## Features

- Interactive Minesweeper board
- Hidden and revealed tiles
- Mine placement
- Flagging tiles
- Win/loss game states
- SFML graphics rendering
- Mouse click event handling

## Technologies Used

- C++
- SFML
- CLion
- Git
- GitHub

## What I Learned

- Object-oriented programming in C++
- Event handling
- 2D grid logic
- Game state management
- Working with external assets
- Debugging graphical programs
